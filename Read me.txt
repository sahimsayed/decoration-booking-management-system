# Decoration Booking Management System

A web-based system to manage decoration bookings for events like weddings, parties, and corporate events. The system allows users to book services, browse the gallery, and the admin can manage requests based on availability.

## Features

- **User Panel**:
  - Browse decoration services (weddings, parties, etc.)
  - Make bookings for events
  - View booking status
  - Contact section

- **Admin Panel**:
  - Manage and approve/reject booking requests
  - View customer details
  - Manage available services

## Technologies Used

- **Frontend**: HTML, CSS, JavaScript, Bootstrap
- **Backend**: PHP, MySQL

## Setup Instructions

### Prerequisites

- A web server with **PHP** support (e.g., XAMPP, WAMP, or similar)
